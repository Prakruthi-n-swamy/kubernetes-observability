# kubernetes-observability
Kubernetes monitoring and observability stack using Prometheus, Grafana, ConfigMaps, Services and Deployments.

## Components

- Deployment
- Service
- ConfigMap
- Prometheus
- Grafana
- ServiceMonitor

## Architecture

```text
Application Pod
      |
      v
Service
      |
      v
Prometheus
      |
      v
Grafana Dashboard
```

## Technologies

- Kubernetes
- Prometheus
- Grafana
- Python
- GitHub Actions

## Monitoring Features

- Pod monitoring
- Service monitoring
- Health checks
- Metrics collection
- Dashboard visualization
